Welcome to a Nand2Tetris companion
=======================================

**Nand2Tetris** is a program designed to learn the whole of modern computing, hardware to software, by building a simulated computer from ground up. Its home is found at `Nand2Tetris <https://nand2tetris.org/>`_ where all the software can be downloaded and materials found.  *simple* and
*intuitive* API.

.. note::

   This project is under active development.
